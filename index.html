Vapour

Vapour is a social gaming platform where users can play single and multiplayer games, as well as communicate via chat.

The platform was developed in a team of 5, for the 2-week long Codeworks Bootcamp Thesis Project.

Vapour was created using the MERN stack, with Redux used for client-side state management. It utilises WebSockets to enable 1-to-1 chat and a discussion forum for logged in users. Vapour also implements a face recognition AI to create custom avatars. The collection of games includes Snake, Pong, Asteroids, Agar.io and more!

TownCrunchr

TownCrunchr is a web-based application simplifying house searching for Londoners!

The app uses public data available in the London Datastore (source) to quantify which borough in London is best for you to live in.

TownCrunchr utilises a dedicated statistical model using Gaussian Distribution to calculate top 3 locations based on salary, target rent, and desired public transportation accessibility. The platform was designed and built with a React frontend, with a combination of custom and Material-UI components. Data about boroughs is accessed from a REST API built entirely from scratch.
