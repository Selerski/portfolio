Vapour

Vapour is a social gaming platform where users can play single and multiplayer games, as well as communicate with one another via chat.

The platform was developed in a team of 5 for the 2-week long Codeworks Bootcamp Thesis Project.

Vapour was created using the MERN stack, with Redux being used for client-side state management. It utilises WebSockets to enable 1-to-1 chat and a discussion forum for logged in users. Vapour also implements a face recognition AI to create custom-made avatars. The collection of games includes Snake, Pong, Asteroids, Agar.io and more!

TownCrunchr

TownCrunchr is a web-based application simplifying house searching for Londoners!

The app uses public data available in the London Datastore (source) to quantify which borough in London is best for you to live in.

TownCrunchr utilises a dedicated statistical model using Gaussian Distribution to calculate top 3 locations based on salary, target rent, and desired public transportation accessibility. The platform was designed and built with a React frontend and a combination of custom-made and Material-UI components. Data is accessed from a REST API built entirely from scratch.
